# todo-list

A simple ToDo-List Python Django WebApp

## Features:

1. Add Tasks
  - Title
  - Description
  - Deadline
  - Label / Category
2. Delete Tasks
  - Delete an individual task
  - Delete multiple selected tasks
  - Delete a complete category / list / labeled tasks
3. Edit Tasks
  - Update all fileds
4. View
  - Search
  - Filter
  - Sort
